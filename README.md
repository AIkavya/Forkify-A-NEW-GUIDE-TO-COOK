## 📌 Overview

**Forkify** is a modern JavaScript application built as part of the **"The Complete JavaScript Course"** on Udemy.

This project demonstrates **real-world JavaScript architecture**, including:

* API integration
* State management
* Modular code structure
* Advanced ES6+ features

---

## ✨ Features

### 🔍 Recipe Search

* Search over **1,000,000+ recipes** using an external API
* Dynamic results with pagination

### 📖 Recipe Details

* Cooking time, servings, ingredients
* Automatically formatted ingredient quantities

### ⚖️ Servings Adjustment

* Increase/decrease servings dynamically
* Ingredient quantities update instantly

### 🔖 Bookmarking

* Save favorite recipes
* Persistent data using **localStorage**

### ➕ Add Your Own Recipes

* Upload custom recipes
* Automatically integrated into search results

### ⚡ Fast & Responsive UI

* Built with modern JavaScript
* Smooth loading experience

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone 
cd forkify
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Start Development Server

```bash
npm start
```

### 4️⃣ Build for Production

```bash
npm run build
```
